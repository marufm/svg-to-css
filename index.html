Info

This tool helps you easily generate css for embedding images with svg/xml data

This tool will:

  • only accept .svg files
  • use the filename as the class name
  • generate a fallback png's in a package for old ie

You will:

  • make sure filenames are class name friendly
  • images are in needed size and colours.
    The tool generates the fallback based on the size/colour attributes of the svg it is given.
